# Kelvin Gum's Developer Portfolio & Research Showcase

This repository contains the source code for Kelvin Gum's personal portfolio website and academic/experimental machine learning notebooks.

## Repository Structure

```
Kelvin-Gum/
├── assets/
│   ├── css/
│   │   └── styles.css       # Custom styling (cyber-glassmorphic dark theme)
│   ├── js/
│   │   └── main.js          # Interactive particles background, scroll-reveal triggers
│   └── images/              # Visual assets, profiles, icons, and animated media
├── ml-research/
│   └── linear_regression_taxi.ipynb # ML Jupyter notebook for NYC Taxi Fare prediction
├── index.html               # Main portfolio interface (responsive semantic HTML)
└── README.md                # Project documentation
```

## Features

- **Cyber-Glassmorphism UI**: Uses HSL variable coloring, blur backdrop filters, custom cursor states, and glowing transitions.
- **Dynamic Particle Network Background**: Features a light-weight HTML5 canvas element rendering dynamically connected nodes reacting to cursor proximity.
- **Responsive Web Design**: Clean layouts using Flexbox and Grid, optimizing performance across desktop, tablet, and mobile browsers.
- **Interactive Notebook Access**: Linking directly to data engineering and training workflows for ML research.

## Technologies Used

* **Core**: HTML5, Vanilla JavaScript (ES6+), Canvas API
* **Styling**: CSS Custom Properties, Flexbox, CSS Grid
* **Machine Learning**: TensorFlow, Pandas, Jupyter Notebooks
* **Design Systems**: Glassmorphism, Google Fonts (`Outfit`, `Montserrat`)
